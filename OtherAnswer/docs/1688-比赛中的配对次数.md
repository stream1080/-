# 1688-比赛中的配对次数

- [题目链接](https://leetcode-cn.com/problems/count-of-matches-in-tournament/)

## 解析

方法一：模拟
- 按照题目，如果是偶数，则除 2
- 如果是奇数，则 -1 除 2

## 参考代码
```Java
class Solution {
    public int numberOfMatches(int n) {
        int count = 0;
        while(n > 1){
            if(n % 2 == 0){
                count += n/2;
                n /= 2;
            }else{
                count += (n-1)/2;
                n = (n-1)/2 + 1;
            }
        }
        return count;
    }
}
```

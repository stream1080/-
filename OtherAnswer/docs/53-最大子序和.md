# 53-最大子序和

- [题目链接](https://leetcode-cn.com/problems/maximum-subarray/)

## 解析

方法一：贪心
- 遍历数组，如果当前序列和大于0，则累加当前元素
- 否则放弃当前序列，以当前元素作为新序列的开始元素
- 更新最大值，遍历结束后返回结果

## 参考代码
```Java
class Solution {
    public int maxSubArray(int[] nums) {
        int max = nums[0];
        int sum = 0;
        for(int num:nums){
            // 当前序列和大于0，累加当前元素
            if(sum>0)
                sum += num;
            else
                // 放弃当前序列，以当前元素作为新序列的开始元素
                sum = num;
            // 更新最大值
            max = Math.max(sum,max);
        }
        return max;
    }
}
```
